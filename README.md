# Honkbal

Start:

```powershell
python D:\honkbal\honkbal\main.py
```

Controls:

- Muis: PCI richten.
- Linksklik of `Space`: normal swing.
- Rechtermuisknop of `X`: power swing.
- `Shift` + klik of `C`: contact swing.
- `H`: compacte/uitgebreide UI wisselen.
- `F11`: fullscreen aan/uit.
- `R`: reset.
- `Esc`: afsluiten.
